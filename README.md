# 查经软件

一个基于Web的圣经经文查询工具，支持智能简拼输入、语音输入和移动端访问。

## 功能特点

### 1. 智能简拼输入
- 输入经卷缩写（如MT），按空格自动选择
- 输入章号，按空格自动加冒号
- 输入节号，按回车查看单节，按空格添加范围
- 章后加z显示整章内容
- 支持大小写通用

### 2. 语音输入
支持多种主流输入法的语音输入格式：
- 豆包输入法：马太福音12章1~3节
- 讯飞输入法：马太福音12章1到3节
- 搜狗输入法：马太福音12章1~3节
- 微信输入法：马太福音十二章一到三节
- 百度输入法：马太福音12章一到三节

### 3. 移动端适配
- 响应式设计，适配各种屏幕尺寸
- 触摸优化，支持手机输入法
- 支持手机语音输入

### 4. 实时显示
- 输入时实时显示经文
- 支持单节、范围、整章查询

## 使用方法

### 键盘输入示例
```
MT 5 3        → 马太福音5:3
MT 5 3 12     → 马太福音5:3-12
MT 5z         → 马太福音第5章全文
CSJ 1 1       → 创世纪1:1
```

### 语音输入示例
```
马太福音12章1到3节      → 马太福音12:1-3
马太福音12章1~3节      → 马太福音12:1-3
马太福音十二章一到三节  → 马太福音12:1-3
```

## 本地运行

### 方法1：Python HTTP服务器
```bash
# 进入项目目录
cd bible

# 启动服务器
python -m http.server 8000

# 访问 http://localhost:8000
```

### 方法2：其他HTTP服务器
可以使用任何HTTP服务器，如Node.js的http-server、PHP的内置服务器等。

**注意**：必须通过HTTP服务器访问，直接打开HTML文件会有CORS限制。

## 部署到GitHub Pages

1. 创建GitHub仓库
2. 上传项目文件
```bash
git init
git add .
git commit -m "初始提交"
git branch -M main
git remote add origin https://github.com/你的用户名/bible-search.git
git push -u origin main
```

3. 启用GitHub Pages
   - 进入仓库设置 → Pages
   - 选择`main`分支作为源
   - 保存后等待部署

4. 访问 `https://你的用户名.github.io/bible-search/`
